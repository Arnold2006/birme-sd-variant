About

A focused variant of BIRME built for Stable Diffusion users who need to crop and resize training images with precise quality controls. Inspired by livelifebythecode/birme-sd-variant.

⚠ Browser Compatibility

Firefox is not supported due to missing imageSmoothingQuality support. Please use Chrome, Edge, or Safari.

Key Features

  • ✓ SD resolution presets (512², 768², 1024², SDXL and more)
  • ✓ Hermite / High / Medium / Low quality presets
  • ✓ Drag-to-reposition and corner-drag to resize crop
  • ✓ Smart focal-point auto-detection via smartcrop.js
  • ✓ JPEG / WebP / PNG output with custom quality
  • ✓ Bulk rename with pattern support
